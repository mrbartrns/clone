# RESTFUL WEBSERVICE
===================
## what is RESTFUL web service?
## RESTFUL WEBSERVCIE의 구조
## GET, POST, PUT, DELETE

### example
    GET/user: 사용자 전체를 조회
    GET/user/273: 273번 사용자를 조회
    POST/user: 사용자를 추가
    DELETE/user/273: 273번 사용자를 삭제

## process
1. 자바스크립트 파일 구성
2. 더미 데이터베이스 구현 > 데이터를 저장하려면 데이터 저장소가 필요
